# Backend Developer

본 과제는 가상의 금융 시장 데이터를 활용하여 종목 정보를 관리하고, 이를 바탕으로 나만의 지수를 구성하여 실시간 시세를 계산하는 백엔드 서비스를 구축하는 것을 목표로 합니다.

## 요구사항

아래의 API를 구현합니다.

### 종목

#### 조회

- 아래의 종목 정보를 제공합니다.
    - 표준코드
    - 단축코드
    - 한글 종목명
    - 한글 종목약명
    - 영문 종목명
    - 상장일
    - 시장구분
    - 증권구분
    - 소속부
    - 주식종류
    - 액면가
    - 상장주식수
- 검색 기능을 제공합니다. (단축코드, 한글 종목명, 한글 종목약명, 영문 종목명)
- 정렬 기능을 제공합니다. (단축코드, 한글 종목약명, 상장일, 상장주식수)
- 필터 기능을 제공합니다. (시장구분, 증권구분, 소속부, 주식종류)

### 지수

#### 조회

- 아래의 지수 정보를 제공합니다.
    - 지수명
    - 영문지수명
    - 기준일
    - 기준지수
    - 구성종목수
- 검색 기능을 제공합니다. (지수명, 영문지수명)
- 정렬 기능을 제공합니다. (지수명, 영문지수명, 기준일, 구성종목수)

#### 추가

- 아래의 지수 정보로 지수를 추가합니다. 기준일은 추가일 당일입니다.
    - 지수명
    - 영문지수명
    - 기준지수
    - 구성종목

#### 수정

- 아래의 정보를 수정하는 기능을 제공합니다.
    - 지수명
    - 영문지수명
    - 구성종목

#### 삭제

- 지수를 삭제하는 기능을 제공합니다.

### 지수 시세

#### 계산

- 아래의 방식을 사용하여 지수의 시세를 계산합니다.
    - `현재 지수 가격 = 현재의 시가 총액 / 기준일의 시가 총액 * 기준지수`
    - 종목이 추가되거나 삭제되더라도 기준일의 시가 총액은 변하지 않습니다.
    - 기준일의 구성 종목들의 시가(open price) 기준 총액의 합을 기준일 시가 총액으로 간주합니다.
- 개별종목의 가격정보는 [외부 API](https://api-recruitment.directional.net/swagger-ui/index.html)에서 가져올 수 있습니다.
- 지수의 시세는 15분마다 계산하여 업데이트합니다.

#### 조회

- 아래의 지수 시세 정보를 제공합니다.
    - 지수명
    - 종가(현재가)
    - 전일대비 등락
    - 전일대비 등락률
    - 시가
    - 고가
    - 저가
- 검색 기능을 제공합니다. (지수명, 영문지수명)
- 정렬 기능을 제공합니다. (지수명, 전일대비 등락률)

## 참고사항

### 조회 API 구현 방식

- 검색 방식: 부분 일치(Contains)를 사용합니다.
- 필터 방식: 다중 조건으로 여러 개를 동시에 적용할 수 있으며 AND 조건으로 동작합니다.

### 종목 정보

- 종목 정보는 `stocks.xlsx` 파일에서 확인 가능합니다.
- 종목 정보를 파일에서 불러오는 기능을 구현하실 필요는 없습니다.

### 외부 API

- `POST /v1/users` API를 통해 비밀번호를 발급받을 수 있습니다.
    - `username`은 이메일주소로 지원서 제출 당시 기재하신 이메일 주소를 사용해주세요.
    - 비밀번호 분실 시 해당 API를 통해 재발급 받을 수 있습니다.
- `POST /auth/token` API를 통해 Access Token을 발급 받을 수 있습니다.
- `GET /v1/prices` API를 통해 가격을 조회할 수 있습니다.
    - 요청시 Access Token을 Authorization 헤더에 포함하여 요청해야 합니다.
        - `Authorization: Bearer <access token>`
    - 제공하는 정보는 아래와 같습니다.
        - `ticker`: 단축코드
        - `open`: 시가
        - `high`: 고가
        - `low`: 저가
        - `close`: 종가(현재가)
        - `change`: 전일대비 등락
    - 해당 시세는 실제 시세가 아니며, 장 운영과 상관없이 1분마다 업데이트됩니다.
